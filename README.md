# Travel Weather Planner

## Beskrivning
Travel Weather Planner är en webbapplikation där man kan söka efter en stad och få:
- Väderprognos för de kommande dagarna  
- Sevärdheter i närheten  
- Karta som visar platsen  

Perfekt för personer som planerar resor eller vill utforska världen hemifrån.

## Användning
1. Skriv namnet på en stad i sökfältet.  
2. Klicka på "Sök" eller tryck Enter.  
3. Se väder, sevärdheter och plats på kartan.

## Tekniker
- JavaScript med AJAX  
- SCSS för styling  
- Leaflet för kartor  
- API:er för väder och platser  
- CSS-animeringar för väderikoner och flygplan
